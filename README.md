# syntropAIbox

Shared utilities for MCP servers providing AST sandboxing, safe execution, and financial tools.

[![PyPI version](https://badge.fury.io/py/syntropAIkit.svg)](https://test.pypi.org/project/syntropAIkit/)
[![Python Versions](https://img.shields.io/pypi/pyversions/syntropAIkit.svg)](https://test.pypi.org/project/syntropAIkit/)

## Description

syntropAIkit is a Python library that provides essential utilities for MCP (Mission Control Protocol) servers, including:

- AST sandboxing
- Safe code execution
- Financial tools and utilities

## Installation

Install from Test PyPI:

```bash
pip install -i https://test.pypi.org/simple/ syntropAIbox
pip install -i https://test.pypi.org/simple/ syntropaibox==X.X.X --extra-index-url https://pypi.org/simple
```

## Requirements

- Python >= 3.10
- Dependencies:
  - hatch >= 1.14.1
  - httpx >= 0.28.1
  - pandas >= 2.3.1
  - pd >= 0.0.4
  - pydantic >= 2.11.7
